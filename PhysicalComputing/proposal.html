Concept and motivation:

I want to build a device which can detect people nearby and work as an intruder alarm, nightlight, or other motion-responsive device.

Anticipated bill of materials:

  • Arduino, used to program device
  • Breadboard for wiring components + cables, resistors, etc.
  • Cardboard for box/container
  • Reactive elements:
  • Buzzer (alarm) or LEDs (nightlight?)
  • Ultrasonic monitor to detect motion

Timeline and contingency plans:

  1. Get basic functionality with working sensors
  2. Test functionality and troubleshoot
  3. Build container and integrate components
  4. Test sensor with container
Contingency: Since I have multiple ways to make the device react, there's a number of fallbacks for responses. For instance, I could use only LEDs or the buzzer if one is difficult to implement.
